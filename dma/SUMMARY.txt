🏆 SPECIFICATIONS
══════════════════════════════════════════════════════════════════════════

Interfaces:
  • AXI4-Lite Slave  - 32-bit configuration interface
  • AXI4-Full Master - 32-bit data transfer interface
  
Channels: 4 independent DMA channels
Priority Levels: 4 (0=lowest, 3=highest)
Burst Sizes: 1, 4, 8, 16, 32, 64, 128, 256 beats
Data Widths: 8-bit, 16-bit, 32-bit
FIFO Depth: 32 entries (expandable)

Register Map:
  Channel spacing: 32 bytes (0x20)
  Registers per channel: 8 (32-bit each)
  - 0x00: Source Address
  - 0x04: Destination Address
  - 0x08: Transfer Length
  - 0x0C: Control Register
  - 0x10: Status Register
  - 0x14: Current Source (RO)
  - 0x18: Current Destination (RO)
  - 0x1C: Remaining Bytes (RO)

🚀 USAGE EXAMPLE
══════════════════════════════════════════════════════════════════════════

Verilog Instantiation:
──────────────────────
dma_top_axi4 #(
    .NUM_CHANNELS(4),
    .ADDR_WIDTH(32),
    .DATA_WIDTH(32),
    .ID_WIDTH(4)
) dma_controller (
    .clk(axi_clk),
    .rst_n(axi_resetn),
    
    // AXI4-Lite Slave (Config)
    .S_AXI_AWADDR(cpu_awaddr),
    .S_AXI_AWVALID(cpu_awvalid),
    // ... other lite signals
    
    // AXI4 Master (Data)
    .M_AXI_AWID(dma_awid),
    .M_AXI_AWADDR(dma_awaddr),
    .M_AXI_AWCACHE(dma_awcache),
    .M_AXI_AWPROT(dma_awprot),
    // ... other full signals
    
    // Interrupts
    .irq_done(dma_done),
    .irq_error(dma_error)
);

Software Configuration:
──────────────────────
// Configure channel 0 for 1KB memory copy
write_dma_reg(DMA_CH0_SRC,    0x80000000);    // Source
write_dma_reg(DMA_CH0_DST,    0x90000000);    // Destination  
write_dma_reg(DMA_CH0_LENGTH, 1024);          // 1KB

// Control: Enable + Start + 16-beat burst + 32-bit + Increment both
uint32_t ctrl = (1<<1) | (1<<0) | (3<<2) | (2<<5) | (1<<7) | (1<<8);
write_dma_reg(DMA_CH0_CTRL, ctrl);

// Wait for completion
while(read_dma_reg(DMA_CH0_STATUS) & 0x1);  // Wait while BUSY

// Check result
if(read_dma_reg(DMA_CH0_STATUS) & 0x4) {
    printf("Error!\n");
} else {
    printf("Success!\n");
}
