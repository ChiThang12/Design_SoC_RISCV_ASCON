Để sử dụng với hardware:
Chạy test trên software để lấy kết quả expected

Chạy cùng test trên hardware (RTL simulation)

So sánh kết quả bằng script

Ví dụ workflow:

bash
# 1. Tạo test vector
python ascon/SW_check/run_auto.py --generate 1 --type aead > test.json

# 2. Extract inputs từ test vector
cat test.json | jq -r '.[0] | .Key, .Nonce, .PT, .AD, .CT'

# 3. Chạy trên software để verify
python ascon/SW_check/run_auto.py --aead \
  --key $(cat test.json | jq -r '.[0].Key') \
  --nonce $(cat test.json | jq -r '.[0].Nonce') \
  --plaintext $(echo $(cat test.json | jq -r '.[0].PT') | xxd -r -p) \
  --ad $(echo $(cat test.json | jq -r '.[0].AD') | xxd -r -p)

# 4. So sánh với hardware output
python ascon/SW_check/run_auto.py --compare "SOFTWARE_OUTPUT_HEX" "HARDWARE_OUTPUT_HEX"
Script này sẽ giúp bạn dễ dàng debug và verify RTL implementation của Ascon!
